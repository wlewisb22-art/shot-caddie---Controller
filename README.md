## What Shot Caddie Does
- Automated shot and cocktail dispensing
- Precision pump timing and calibration
- Supports multiple spirits and mixers
- Designed for home bars, events, and small venues
- Raspberry Pi + microcontroller based control system

## Technology Overview
- Raspberry Pi (web UI and control logic)
- STM32 microcontrollers (motor control)
- Peristaltic pumps
- Web-based UI (Flask / ASP.NET prototypes)
- SQL-backed configuration and calibration
